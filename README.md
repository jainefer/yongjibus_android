# YongJiBus
### 명지대학교 자연 캠퍼스(용인) 셔틀버스 시간표 어플리케이션

## 프로젝트 목적

- 학생들이 교내 및 교외 셔틀 시간을 확인 하기 위해서는 학교 홈페이지를 통해 여러 번의 탐색을 통해 확인해야하는 번거러움이 있다.
- 해당 번거로움을 해소하기 위해 프로젝트 진행
- 최대한 간편하게 시간표를 확인할 수 있는 것이 목표

## 주요기능

- 명지대역 및 시내 셔틀 시간표
- 기흥역 셔틀 시간표
- 진입로 빨간버스 도착 예정 시간 표시

## 개발 스택
- AndroidStudio
- OpenAPI (공공데이터)
- MVVM
